* 34.7% children under 5 are stunted.

## Installation
 ## Data Intepretation and Model
* Clone the repo
```sh
   $ git clone https://github.com/Shakileash5/Food_security.git
```
```sh
   $ cd Food_security
```
* Create virtualenv
```sh
   $ virtualenv venv
```
* For Linux
```sh
   $ source ./venv/bin/activate
```
* For windows
```sh
   $ cd venv/Scripts/
```
```sh
   $ activate
```
* Install dependencies
```sh
   $ pip install -r requirements.txt
```
